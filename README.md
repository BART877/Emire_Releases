## 📖 About The App

**Emire Manga Reader & Editor** is a high-performance desktop application designed for manga enthusiasts, translators, and editors. Built with security and user experience at its core, Emire provides a seamless environment for reading manga while offering powerful tools for professional-grade editing and scanlation.

## ✨ Key Features

- **Advanced Manga Reader:** Enjoy a smooth, customizable reading experience with support for high-resolution images, multiple viewing modes (single-page, double-page spread, vertical scroll), and optimized hardware acceleration.
- **Built-in Editor Toolkit:** Features a specialized suite of editing tools designed specifically for manga scanlation, including text cleaning, typesetting, and image adjustments.
- **Secure Licensing & Authentication:** Protected by an enterprise-grade authentication system ensuring secure access through a robust subscription model.
- **Over-The-Air (OTA) Updates:** Seamless auto-update system that detects, safely downloads, and updates the desktop client with zero hassle.
- **Cloud-Synced Profiles:** User preferences, reading history, and editor configurations are securely synced to the cloud.

## 🚀 Download & Installation

To get the latest version of Emire:

1. Navigate to the [**Releases**](https://github.com/BART877/Emire_Releases/releases) page on the right side of this repository.
2. Under the latest version, download the `EmireSetup_vX.X.exe` file.
3. Run the installer and follow the on-screen instructions.
4. Launch Emire, log in with your licensed account, and start reading!
